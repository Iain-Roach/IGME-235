Creating Experiences to Lose Yourself in

About Me

I am a Game Developer/Designer from Anchorage, Alaska. I enjoy constructing worlds that allow people to lose themselves in a new world. In my spare time you can find me delving through Dungeons and Dragons or out scoring goals in soccer.

Projects

Island Explorer

Small rpg using Javascript and the browser DOM made in my Game Web Tech Class. You explore the island fighting monsters in turn-based combat and finding items that unlock new abilities. This project, I implemented ES6 Classes into my Javascript, simple map navigation between levels, and how to use spritesheets, CSS and the DOM to project a game world.

Asteroids

Recreation of Asteroids in Unity 2D and C#, Move the ship using the arrow keys, and fire missiles with the spacebar. I learned the basics of collision detection, and 2D movment in Unity

D&D SpellBook

Small spell search tool for D&D 5th edition SRD. Uses the 5th Edition SRD API as well as GraphQL to allow the user to filter levels, and school of spells, as well as searching them via their name. Built using HTML and javascript. This project was used to help me learn how to use API's and I also started learning GraphQL in the process.
